> **Example error message:**<br/>
**blog** → **post.md** frontmatter does not match collection schema.<br/>
"title" is required.<br/>
"date" must be a valid date.

## 哪里发生了错误？
在`src/content/`目录下的Markdown或MDX条目与其集合模式不匹配。
确保所有必需的字段都存在，并且所有字段都是正确的类型。
请检查`src/content/config.*`文件中的集合模式。
更多信息，请参见[内容集合](/zh-ch/guides/content-collections/)
